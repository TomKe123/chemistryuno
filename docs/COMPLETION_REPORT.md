## 💡 核心功能

### 1️⃣ 游戏初始化
```
→ 创建/加入游戏
→ 自动分配玩家ID
→ 每人初始10张牌
→ 随机洗牌
```

### 2️⃣ 出牌流程
```
→ 玩家点击卡牌
→ 系统查询可组成物质
→ 浮窗显示物质列表
→ 玩家选择物质
→ 验证反应
→ 更新游戏状态
```

### 3️⃣ 特殊卡牌
```
→ He/Ne/Ar/Kr: 反转方向
→ Au: 跳过下家
→ +2: 下家摸2张
→ +4: 下家摸4张
```

### 4️⃣ 游戏结束
```
→ 玩家清空手牌
→ 宣布获胜者
→ 计算得分
→ 返回游戏大厅
```
