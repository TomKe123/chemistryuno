## 🔧 前置要求

在开始之前，请确保你的系统已安装：

### 必需软件
- **Node.js** >= 14.0
  - 检查版本：`node --version`
  - 下载地址：[nodejs.org](https://nodejs.org/)
  
- **pnpm** >= 8.0（推荐使用）
  - 检查版本：`pnpm --version`
  - 安装命令：`npm install -g pnpm`
  - 官方文档：[pnpm.io](https://pnpm.io/)

> 💡 **为什么使用pnpm？**
> - ⚡ 比npm快2-3倍
> - 💾 节省磁盘空间（共享依赖）
> - 🔒 更严格的依赖管理

## 📦 安装步骤

### 1. 克隆或下载项目

```bash
# 如果使用Git
git clone <仓库地址>
cd chemistryuno

# 或直接解压下载的ZIP文件
```

### 2. 安装依赖

```bash
# 使用pnpm（推荐）
pnpm install

# 或使用npm
npm install
```

这个命令会自动安装前端和后端的所有依赖包。

> 📝 **注意**：首次安装可能需要几分钟，请耐心等待。

## 🎬 启动应用

### 开发模式（推荐用于学习和测试）

```bash
# 启动开发服务器（包括前端和后端）
pnpm start

# 或使用热重载模式
pnpm run dev
```

启动成功后，你会看到：
```
Server running on http://localhost:5000
React app running on http://localhost:3000
```

### 访问应用

打开浏览器访问：
- **游戏界面**：http://localhost:3000
- **管理面板**：http://localhost:3000/admin
- **服务器状态**：http://localhost:5000

## 🎮 开始游戏

### 电脑端游戏

#### 作为房主（创建游戏）

1. **打开游戏界面**
   - 浏览器访问 `http://localhost:3000`

2. **输入玩家名称**
   - 在"玩家名称"输入框中输入你的昵称
   - 昵称会被自动记住

3. **创建房间**
   - 点击"创建游戏"标签
   - 选择玩家数量（2-12人，推荐4人）
   - 点击"创建游戏"按钮

4. **分享房间**
   - 房间创建后会显示：
     - 房间号（6位数字）
     - 二维码（可扫码加入）
   - 分享给其他玩家

5. **等待玩家加入**
   - 等待其他玩家加入房间
   - 人数足够后点击"开始游戏"

#### 作为玩家（加入游戏）

1. **打开游戏界面**
   - 浏览器访问 `http://localhost:3000`

2. **输入信息**
   - 输入你的玩家名称
   - 输入房间号（由房主提供）

3. **加入房间**
   - 点击"加入游戏"按钮
   - 等待房主开始游戏

### 📱 移动端游戏

详细的移动端访问指南，请参考：[移动端访问指南](MOBILE_ACCESS_GUIDE.md)

#### 快速步骤

1. **确保设备在同一WiFi网络**
   - 手机和电脑需要在同一个局域网

2. **获取电脑IP地址**
   
   **Windows系统：**
   ```powershell
   ipconfig
   ```
   查找"IPv4 地址"（通常是 192.168.x.x）

   **macOS/Linux系统：**
   ```bash
   ifconfig
   # 或
   ip addr
   ```

3. **手机访问**
   - 打开手机浏览器
   - 访问：`http://[电脑IP]:3000`
   - 例如：`http://192.168.1.100:3000`

4. **扫码或输入房间号加入游戏**

## 🎯 游戏玩法

### 基本规则

1. **初始手牌**：每位玩家开局获得10张牌

2. **出牌规则**：
   - 上家打出一个化学物质
   - 你需要打出能与该物质**反应**的物质
   - 系统会自动显示你当前手牌能组成的所有物质

3. **如何出牌**：
   - 点击手牌中的元素卡
   - 系统弹出可组成的物质列表
   - 选择能与场上物质反应的物质
   - 确认打出

4. **无法出牌**：
   - 点击"摸牌"按钮
   - 自动摸2张牌
   - 回合结束

5. **特殊卡牌**：
   - **He/Ne/Ar/Kr**（稀有气体）：反转游戏方向
   - **Au**（金）：跳过下一位玩家
   - **+2**：下家摸2张牌
   - **+4**：下家摸4张牌

6. **获胜条件**：
   - 最先打完所有手牌的玩家获胜

### 示例回合

```
场上物质：AgNO3（硝酸银）
你的手牌：Na, Cl, H, O

1. 点击 Na 和 Cl 卡牌
2. 系统显示可组成：NaCl（氯化钠）
3. 系统检测：AgNO3 + NaCl → 可以反应 ✓
4. 确认打出 NaCl
5. 下一位玩家需要打出能与 NaCl 反应的物质
```

## ❓ 常见问题

### 启动问题

**Q: 启动时提示端口被占用**
```
Error: Port 3000 is already in use
```

**A:** 修改端口或关闭占用程序
```bash
# 查找占用端口的程序
# Windows
netstat -ano | findstr :3000

# macOS/Linux
lsof -i :3000

# 或修改端口（在 client/package.json 中设置 PORT 环境变量）
```

**Q: 依赖安装失败**

**A:** 尝试以下方法：
1. 清除缓存：`pnpm store prune` 或 `npm cache clean --force`
2. 删除 `node_modules` 和 `pnpm-lock.yaml`
3. 重新安装：`pnpm install`
4. 使用淘宝镜像：`pnpm install --registry=https://registry.npmmirror.com`

### 游戏问题

**Q: 手机无法连接游戏**

**A:** 检查以下几点：
1. 手机和电脑是否在同一WiFi网络
2. 防火墙是否允许端口 3000 和 5000
3. IP地址是否正确（不要使用 127.0.0.1 或 localhost）
4. 参考 [移动端访问指南](MOBILE_ACCESS_GUIDE.md)

**Q: 断线后无法重新加入**

**A:** 系统支持断线重连：
1. 使用相同的玩家名称
2. 输入相同的房间号
3. 点击"加入游戏"即可恢复

**Q: 找不到能打出的物质**

**A:** 可能情况：
1. 你的手牌确实无法组成能反应的物质 → 摸牌
2. 系统数据库中没有该反应 → 可通过管理面板添加

### 配置问题

**Q: 如何修改游戏规则？**

**A:** 访问管理面板
1. 打开 `http://localhost:3000/admin`
2. 输入管理员密码
3. 添加/修改化学反应规则
4. 保存配置

详见：[管理面板指南](ADMIN_PANEL_GUIDE.md)

**Q: 如何修改初始手牌数量？**

**A:** 编辑 `config.json` 文件中的配置，或通过管理面板修改。
