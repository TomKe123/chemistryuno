# 🚀 快速开始指南

欢迎使用化学UNO游戏！本指南将帮助你快速上手。

## 📋 前置要求

- **Node.js**: >= 18.0.0
- **pnpm**: >= 8.0.0

## 安装 pnpm

如果还没有安装 pnpm：

```bash
npm install -g pnpm
```

## 🎮 开始游戏

### 方法一：一键启动（最简单）

```bash
node start.js
```

这个脚本会自动：
1. 检查 pnpm 是否已安装
2. 首次运行时自动安装依赖
3. 启动开发服务器

### 方法二：使用 pnpm 命令

```bash
# 1. 安装依赖（首次运行）
pnpm install

# 2. 启动开发服务器
pnpm run dev
```

## 🌐 访问游戏

启动后，在浏览器中打开：

- **游戏主页**: http://localhost:3000
- **管理面板**: http://localhost:3000/admin

## 🎯 游戏流程

### 1. 创建房间
- 输入昵称
- 点击"创建房间"
- 记住房间号并分享给其他玩家

### 2. 加入房间
- 其他玩家输入昵称和房间号
- 点击"加入房间"

### 3. 开始游戏
- 等待所有玩家加入
- 房主点击"开始游戏"按钮

### 4. 游戏玩法
1. 每位玩家初始有10张元素牌
2. 当轮到你时，选择元素组成物质
3. 物质必须能与上家打出的物质发生化学反应
4. 无法打出时自动摸2张牌
5. 最先打完所有牌的玩家获胜

## 📱 手机游戏

1. 确保手机和电脑在同一WiFi网络
2. 启动游戏后，服务器会显示二维码
3. 用手机扫描二维码即可访问

详见[移动端访问指南](MOBILE_ACCESS_GUIDE.md)

## 🔧 管理面板

访问 http://localhost:3000/admin 进入管理面板。

**设置管理员密码**：

1. 编辑 `client/.env.production` 文件
2. 设置 `REACT_APP_ADMIN=your_password_here`
3. 重新构建前端：`pnpm run build:client`

在管理面板中可以：
- 修改卡牌数量
- 添加/删除物质
- 编辑化学反应
- 调整游戏规则

详见[管理面板指南](ADMIN_PANEL_GUIDE.md)

## 🛠️ 常见问题

### 端口被占用

如果3000或5000端口被占用，可以修改：

**前端端口**：编辑 `client/.env`
```env
PORT=3001
```

**后端端口**：编辑 `server/index.ts` 中的端口号

### 依赖安装失败

```bash
# 清理缓存并重新安装
pnpm store prune
pnpm install
```

### 游戏无法连接

1. 检查防火墙设置
2. 确保前后端都正常启动
3. 查看浏览器控制台的错误信息
