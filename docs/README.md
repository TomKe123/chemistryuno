## 🎮 游戏规则简述

1. **目标**：最先打完手牌获胜
2. **出牌**：打出能与上家物质发生化学反应的物质
3. **特殊牌**：
   - Au（金）：跳过下家
   - He/Ne/Ar/Kr（惰性气体）：反转方向
   - +2/+4：下家摸牌

详细规则见[主页](../README.md)
