# 🎮 化学UNO - 游戏规则速查

快速查阅游戏规则和玩法。

## 🎯 游戏目标

**最先打完所有手牌的玩家获胜！**

## 👥 游戏人数

- **最少**: 2人
- **最多**: 12人
- **推荐**: 4人

|------|------|------|

## 📖 基本规则

### 开局

1. 每位玩家发 **10张牌**
2. 从牌堆翻出第一张牌
3. 房主点击"开始游戏"
4. 按顺序开始出牌

### 出牌规则

#### 普通回合

1. **查看场上物质**
   - 上一位玩家打出的物质显示在中央

2. **选择元素卡**
   - 点击手中的元素卡
   - 可以选择多张元素卡

3. **选择物质**
   - 系统弹出可组成的物质列表
   - 选择一个能与场上物质反应的物质

4. **确认打出**
   - 系统自动验证反应
   - 成功则出牌，失败则提示

5. **轮到下家**

#### 无法出牌

如果无法打出能反应的物质：

1. 点击"摸牌"按钮
2. 自动摸 **2张牌**
3. 回合结束，轮到下家

### 特殊卡规则

#### 稀有气体（He/Ne/Ar/Kr）

**效果**：反转游戏方向

```
原方向: A → B → C → D
打出He后: D → C → B → A
```

**使用方法**：
- 直接打出（不需要组成物质）
- 立即反转方向
- 继续游戏

#### 金（Au）

**效果**：跳过下一位玩家

```
当前玩家: A
打出Au后: 跳过B，轮到C
```

**使用方法**：
- 直接打出
- 下家无法出牌，直接轮到下下家

#### 摸牌卡（+2/+4）

**效果**：指定下家摸牌

```
打出+2: 下家摸2张牌
打出+4: 下家摸4张牌
```

**使用方法**：
- 直接打出
- 下家必须摸牌
- 摸牌后回合结束

## 🧪 化学反应规则

### 什么是"能反应"？

两个物质能反应，需要满足：

1. 在配置文件的反应规则中有记录
2. 或者反应规则中有对应的反应

**示例**：

```
场上物质: AgNO3（硝酸银）
你的物质: NaCl（氯化钠）

系统检查:
AgNO3 + NaCl → AgCl↓ + NaNO3 ✓
可以打出！
```

### 常见反应类型

#### 1. 复分解反应（沉淀）

```
AgNO3 + NaCl → AgCl↓ + NaNO3
BaCl2 + H2SO4 → BaSO4↓ + 2HCl
```

#### 2. 酸碱中和

```
HCl + NaOH → NaCl + H2O
H2SO4 + 2KOH → K2SO4 + 2H2O
```

#### 3. 置换反应

```
CuSO4 + Fe → FeSO4 + Cu
Zn + H2SO4 → ZnSO4 + H2↑
```

#### 4. 氧化还原

```
Fe2O3 + 3CO → 2Fe + 3CO2
2KClO3 → 2KCl + 3O2↑
```

### 物质选择

系统会自动显示你能组成的所有物质：

```
你的手牌: H, H, O, Na, Cl

可组成物质:
- H2O（水）
- HCl（盐酸）
- NaCl（氯化钠）
- NaOH（氢氧化钠）
```

选择能与场上物质反应的即可。

## 🎲 游戏策略

### 基础策略

1. **优先出常见元素**
   - H、O等常见元素容易组成多种物质
   - 保留稀有元素应对特殊情况

2. **注意特殊卡时机**
   - 反转卡：在不利方向时使用
   - 跳过卡：跳过强势玩家
   - 摸牌卡：让领先者摸牌

3. **记忆已打出的卡牌**
   - 判断其他玩家可能有的元素
   - 预测能打出的物质

4. **快速出牌**
   - 手牌越多越不利
   - 尽快清空手牌

### 进阶技巧

1. **卡牌计数**
   - 记录哪些元素已经打完
   - 判断某些反应是否还能发生

2. **物质选择**
   - 选择常见反应物
   - 让下家更容易接牌（或更难）

3. **元素组合**
   - 保留能组成多种物质的元素组合
   - 提高出牌灵活性

## 🏆 胜利条件

### 获胜

- 第一个打完所有手牌的玩家获胜
- 游戏立即结束
- 显示获胜者名称

### 游戏结束后

1. 显示获胜者信息
2. 可以选择：
   - 返回大厅创建新游戏
   - 退出游戏

## 📱 操作说明

### 电脑端

1. **出牌**：鼠标点击卡牌
2. **选择物质**：点击弹出列表中的物质
3. **摸牌**：点击"摸牌"按钮
4. **查看信息**：鼠标悬停在卡牌上

### 移动端

1. **出牌**：触摸点击卡牌
2. **选择物质**：触摸选择物质
3. **摸牌**：触摸"摸牌"按钮
4. **查看信息**：长按卡牌

## 🎯 游戏示例

### 完整回合示例

```
回合1: 玩家A（你）
场上: H2O（水）
手牌: H, Cl, Na, O

步骤:
1. 思考: 什么能与H2O反应？
2. 选择: H + Cl → HCl（盐酸）
3. 检查: H2O + ... 无法反应 ✗
4. 重选: Na + O → Na2O（氧化钠）
5. 检查: H2O + Na2O → 2NaOH ✓
6. 打出: Na2O
7. 场上变为: Na2O

回合2: 玩家B
场上: Na2O
...
```

### 特殊卡使用示例

```
情况1: 游戏方向对你不利
你的手牌: He, H, O, C
操作: 打出He，反转方向

情况2: 下家快要赢了（只剩2张牌）
你的手牌: +2, Au, H, O
操作: 打出+2，让下家摸牌

情况3: 你想跳过某个玩家
你的手牌: Au, H, O
操作: 打出Au，跳过下家
```

## ⚠️ 注意事项

1. **不考虑化学计量数**
   - 只考虑反应物和生成物
   - 不需要配平方程式

2. **特殊卡可以随时打出**
   - 不需要满足反应条件
   - 直接产生效果

3. **摸牌后回合结束**
   - 即使摸到能出的牌也不能出
   - 必须等下次轮到你

4. **系统自动验证**
   - 只能打出有效的物质组合
   - 反应验证由系统自动完成
