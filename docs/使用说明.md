# 🎮 化学UNO - 使用说明

## 🚀 快速启动

### Windows 用户（最简单）⭐推荐
```
双击 start.bat 文件
```

### Linux/Mac 用户
```bash
# 首次运行，添加执行权限
chmod +x start.sh

# 启动游戏
./start.sh
```

### 跨平台方式（所有系统）
```bash
# 使用 Node.js 脚本
node start.js

# 或使用 pnpm 命令
pnpm run dev
```

## 🌐 访问地址

启动后访问：
- **游戏主页**: http://localhost:3000
- **管理面板**: http://localhost:3000/admin（密码在 `client/.env.production` 中配置）

## 📦 生产部署

### Windows 用户
```
双击 deploy.bat 文件
```

### Linux/Mac 用户
```bash
# 首次运行，添加执行权限
chmod +x deploy.sh

# 运行部署
./deploy.sh
```

### 跨平台方式
```bash
# 使用 pnpm 命令
pnpm run deploy
```

部署后访问：
- **游戏**: http://localhost:4000
- **后端API**: http://localhost:5000

## 💡 常见问题

### 1. 提示 "pnpm 不是内部或外部命令"
```bash
# 安装 pnpm
npm install -g pnpm
```

### 2. 端口被占用
- 修改前端端口：编辑 `client/.env`
- 修改后端端口：编辑 `server/index.ts`

### 3. 依赖安装失败
```bash
# 清理并重新安装
pnpm store prune
pnpm install
```
